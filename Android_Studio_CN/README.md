使用说明：
Android Studio 中文包使用方法：
1、点击下载得到Android Studio 中文包
2、将C:\Program Files (x86)\Android Studio\lib目录下的resources_en.jar文件复制出来，并更名为resources_cn.jar
3、双击打开resources_cn.jar（PS：不是解压），将下载的汉化包zh_CN目录下的所有文件拖到刚才打开的resources_cn.jar文件内的messages目录中，并保存
4、将resources_cn.jar文件复制回C:\Program Files (x86)\Android Studio\lib目录
5、OK！
